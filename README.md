# Discord-bot

KITの一年生のDiscordのチャンネル整理をするために開発したbotです。

# Note

科目ごとのチャンネルを自動で生成し、履修登録した人のみ通知が来るように設定します。

具体的には科目ごとに役職を作り、通知を受け取りたい人は役職付与メッセージを送信することで役職が付与されます。

Carl-bot(外部サービス)を利用すると役職付与はリアクションによってすることもできます。
